Semantic HTML page set-up cheatsheet

A quick reference to all the basic building blocks for a semantic HTML webpage.
For more information about HTML and HTML tags, visit W3schools.com.
Lean how to code on Codecademy!

Basic HTML elements

These are the basic building blocks needed to structure a webpage created with HTML.

Tag Name Description
<! DOCTYPE html> HTML document type Declaration All HTML5 documents must start with a declaration, so the browser can interpret it as the current HTML standard. This is a self-closing tag.
<html></html> HTML tag The Hyper Text Markup Language tag comes in a pair at the start and end of an HTML page.
<head></head> Head element This element is a container for metadata (data about data), it is placed above the <body> element.
<title></title> Title element This tag is required in all HTML documents and it defines the title of the document, which is used to display a title in the browser toolbar/ favorites/ search-engine results. It is placed inside the <head> element.
<link rel="stylesheet" type="text/css" href="***.css"> Link to stylesheet In order to apply external CSS to a page, link to the stylesheet in the <head> element.
<body></body> Body element All data inside this element will be displayed by the browser.
<h1></h1> Header element The <h1> to <h6> tags are used to define HTML headings, where <h1> defines the most important/ largest heading and <h6> defines the least important/ smallest heading.
<p></p> Paragraph element This tag defines a paragraph.
<a></a>

<a href="***.html">This is a link</a>		 Anchor element This tag defines a hyperlink, which is used to link from one location to another. Its most important attribute is the 'href' (hyperlink reference) attribute, which indicates the link's destination.
<img>

<img src="***.jpg" alt="blank image">		 Image element This self-closing tag defines the location of an image in an HTML page, it has two required attributes: 'src' and 'alt'.

Semantic HTML elements

A semantic element clearly describes its meaning to both the browser and the developer.

Tag Name Description
<header></header> Header element The <header> element specifies a header for a document or section, it should be used as a container for introductory content.
<nav></nav> Navigation element The <nav> element defines a set of navigation links.
<main></main> Main element This tag specifies the main content of a document. The content inside it should be unique to the document.
<section></section> Section element The <section> element defines a section in a document.
<article></article> Article element The <article> element specifies independent, self-contained content.
<aside></aside> Aside element This tag defines some content aside from the content it is placed in, which should be related to the surrounding content.
<figure></figure> Figure element An image and a caption can be grouped together in a <figure> element. Place the <img> tag inside of this element.
<figcaption></figcaption> Figure caption element The purpose of a caption is to add a visual explanation to an image. Place it under the <img> tag inside the <figure> element.
<footer></footer> Footer element This tag defines a footer for a document or section, it should contain information about its containing element.
